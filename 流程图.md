下面是这张 **TEC-8 硬布线控制器流程图** 的文字版整理。

# 一、入口与模式选择

```text
CLR
 ↓
根据 SWC SWB SWA 选择工作模式
```

| SWC SWB SWA | 模式     |
| ----------- | -------- |
| 000         | 取指     |
| 001         | 写存储器 |
| 010         | 读存储器 |
| 011         | 读寄存器 |
| 100         | 写寄存器 |

------

# 三、指令执行部分

根据指令寄存器高 4 位：

```text
IR7 ~ IR4
```

进行指令译码。

| IR7~IR4 | 指令 |
| ------- | ---- |
| 0000    | NOP  |
| 0001    | ADD  |
| 0010    | SUB  |
| 0011    | AND  |
| 0100    | INC  |
| 0101    | LD   |
| 0110    | ST   |
| 0111    | JC   |
| 1000    | JZ   |
| 1001    | JMP  |
| 1110    | STP  |

------

## 1. 0000：NOP

### W1

```text
LIR
PCINC
SHORT
```

含义：不执行有效运算，直接取下一条指令。

------

## 2. 0001：ADD

### W1

```text
S = 1001
CIN
ABUS
DRW
LDZ

LIR
PCINC
SHORT
```

含义：执行加法，结果写回寄存器，更新 Z 标志，同时取下一条指令。

------

## 3. 0010：SUB

### W1

```text
S = 0110
ABUS
DRW
LDZ
LDC

LIR
PCINC
SHORT
```

含义：执行减法，结果写回寄存器，更新 Z、C 标志，同时取下一条指令。

------

## 4. 0011：AND

### W1

```text
M
S = 1011
ABUS
DRW
LDZ

LIR
PCINC
SHORT
```

含义：执行按位与运算，结果写回寄存器，更新 Z 标志，同时取下一条指令。

------

## 5. 0100：INC

### W1

```text
S = 0000
ABUS
DRW
LDZ
LDC

LIR
PCINC
SHORT
```

含义：执行自增运算，结果写回寄存器，更新 Z、C 标志，同时取下一条指令。

------

## 6. 0101：LD

LD 指令分两拍执行。

### W1

```text
M
S = 1010
ABUS
LAR
```

### W2

```text
MBUS
DRW

LIR
PCINC
```

含义：

```text
W1：把地址送入地址寄存器 LAR
W2：从存储器读数据，经 MBUS 写回寄存器，并取下一条指令
```

------

## 7. 0110：ST

ST 指令也分两拍执行。

### W1

```text
M
S = 1111
ABUS
LAR
```

### W2

```text
M
S = 1010
ABUS
MEMW

LIR
PCINC
```

含义：

```text
W1：把目标地址送入地址寄存器 LAR
W2：把数据写入存储器，并取下一条指令
```

------

## 8. 0111：JC

JC 是根据进位标志 C 判断是否跳转。

### C = 0，不跳转，W1

```text
LIR
PCINC
SHORT
```

### C = 1，跳转

#### W1

```text
PCADD
```

#### W2

```text
LIR
PCINC
```

含义：

```text
C = 0：不跳转，直接顺序取下一条指令
C = 1：执行 PC 相对跳转，然后取下一条指令
```

------

## 9. 1000：JZ

JZ 是根据零标志 Z 判断是否跳转。

### Z = 0，不跳转，W1

```text
LIR
PCINC
SHORT
```

### Z = 1，跳转

#### W1

```text
PCADD
```

#### W2

```text
LIR
PCINC
```

含义：

```text
Z = 0：不跳转，顺序执行
Z = 1：执行 PC 相对跳转
```

------

## 10. 1001：JMP

### W1

```text
M
S = 1111
ABUS
LPC
```

### W2

```text
LIR
PCINC
```

含义：无条件跳转，把目标地址装入 PC。

------

## 11. 1110：STP

### W1

```text
STOP
```

含义：停机。





# 补充

在 TEC-8 实验系统中，由于按下复位按钮 CLR 后，指令寄存器 IR 被复位为00H，因此，无论存储器中写入什么测试程序，在图 2.2 所示的流程图中，第 1条被执行的指令的代码总是 00H。我们规定，指令代码为 00H 的指令是条 nop 指令
